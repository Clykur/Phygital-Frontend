Build a modern, aesthetic, premium website for a startup called:

“Phygital Student Library Network”

The website should feel like a futuristic digital library — clean, minimal, and high-end. It should NOT look like a template.

-----------------------------------

🎨 DESIGN STYLE
- Minimal, elegant, premium
- Light + soft dark tones (white, soft gray, deep blue)
- Smooth animations on scroll
- Clean typography (Inter or Sora)
- Glassmorphism cards + subtle shadows
- Lots of whitespace
- Soft glowing elements (like digital library feel)

-----------------------------------

🧠 EXPERIENCE
The whole website should feel like:
“Walking inside a modern digital library”

- Books, shelves, and knowledge-inspired visuals
- Smooth transitions between sections
- Calm, focused, intellectual vibe (not flashy)

-----------------------------------

📄 SECTIONS (LANDING PAGE)

1. HERO
- Title: “Access Knowledge Without Owning It”
- Subtitle: Short explanation of phygital library network
- Buttons: [Explore Network] [Get Started]

2. PROBLEM
- Students: Expensive textbooks, unused after semester
- Colleges: Outdated, underused libraries
- Use clean icons + minimal text

3. SOLUTION
- Explain Phygital model:
  Physical hubs + Digital app + Network
- Use simple diagram or cards

4. HOW IT WORKS
- 4 steps:
  Search → Borrow/Buy → Raise Query → Pickup at Hub
- Use clean horizontal cards

5. FEATURES
- Live inventory
- QR scanning
- P2P marketplace
- Smart routing

6. BUSINESS MODEL (simple)
- Subscription ₹199/month
- Book sales
- P2P transactions

7. IMPACT
- Save money for students
- Better libraries for colleges
- Sustainable book usage

8. FINAL CTA
- “Join the Library Network”
- Buttons for Students / Colleges

-----------------------------------

📄 EXTRA PAGES

- Student Experience (dashboard style UI)
- College Partners (benefits)
- P2P Marketplace (simple listings UI)
- About (vision + mission)

-----------------------------------

⚙️ TECH
- React + Tailwind
- Mobile-first design
- Smooth animations (Framer Motion style)
- Use good quality images (books, libraries, students)

-----------------------------------

✨ IMPORTANT
- Keep it clean and premium
- Avoid clutter
- Avoid template-like layout
- Focus on spacing, typography, and smooth UX
- Make it feel like a real startup product

-----------------------------------

OUTPUT:
A clean, aesthetic, premium website with smooth scrolling and modern UI.
